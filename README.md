# Mini éditeur de texte (ACO)

Projet académique réalisé dans le cadre de la MIAGE (ISTIC – Université de Rennes).

## Objectif
Implémenter un mini éditeur de texte en Java en appliquant les principes de l’**Architecture de Conception Objet (ACO)**, notamment les design patterns :
- Command
- Memento

## Fonctionnalités
- Insertion de texte
- Copie / Coller
- Couper / Supprimer
- Modification de la sélection
- Annulation / Restauration de l’état (undo)

## Technologies
- Java
- JavaFX
- JUnit
